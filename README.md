#Sistema de Leilões 
## Sobre o sistema
Sistema criado para gerenciamento de uma casa de leilões, o progama conta com sistema de cadastro e venda.

## Tecnologias
-Java
-MySQL
-GitBub
-Git

## Funcionalidades
-Cadastro de Produtos
-Armazenamento de dados no banco de dados
-Vender produtos e registro de venda no banco de dados
-Listagem de produtos na tabela
-Listagem de produtos vendidos na tabela de produtos vendidos
